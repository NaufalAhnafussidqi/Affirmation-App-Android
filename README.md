# Affirmations App Android

Aplikasi Android menggunakan Jetpack Compose untuk menampilkan daftar afirmasi yang dapat di-scroll menggunakan komponen LazyColumn dan Card.

## Fitur
- Menampilkan daftar data menggunakan LazyColumn
- Menggunakan Card untuk setiap item
- Menggunakan resources gambar
- Menggunakan resources string
- Mendukung scrolling yang lancar
